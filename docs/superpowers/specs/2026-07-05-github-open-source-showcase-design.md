# Mineradio GitHub 开源展示优化设计

## 目标

在现有 `zzyxiangnian-star/Mineradio` 仓库基础上发布 v1.3.5 更新，使仓库首页能够准确、清晰地展示 Mineradio 的产品价值、真实功能、技术结构和运行效果，并让开发者可以按文档完成安装、启动与构建。

本次只描述代码中已经存在的能力。尚未实现或无法验证的内容不作为核心功能宣传。

## 项目定位

Mineradio 是一款基于 Electron 的沉浸式桌面音乐播放器，围绕多平台音乐接入、AI 音乐推荐、桌面歌词、动态壁纸、实时音频可视化和 3D 歌单浏览，提供从搜索、登录、播放到视觉呈现的完整桌面体验。

主要面向开源用户、桌面应用开发者，以及希望了解 Electron、多媒体交互、第三方音乐服务接入和 AI 推荐工程实践的面试官或技术评审者。

## 技术架构表达

README 架构图反映以下真实数据流：

- Electron 主进程负责窗口、托盘、登录弹窗、桌面歌词、壁纸窗口、全局快捷键与本地服务生命周期。
- HTML/CSS/JavaScript 渲染层负责播放器界面、Music Soul、歌词舞台、Three.js/GSAP 视觉效果和用户交互。
- Node.js 本地服务负责静态资源、音乐服务代理和业务接口。
- 网易云音乐、QQ 音乐、酷狗音乐、汽水音乐及本地文件构成音乐来源。
- MiMo 或兼容 Chat Completions 接口用于 AI DJ、音乐画像和推荐能力；API Key 只从环境变量或用户本地配置读取。
